Personal Portfolio Website

A front-end project focuses on developing the user-facing part of a website or application. This area involves crafting the visual and interactive elements that users engage with directly. It includes designing the layout, choosing color schemes, and ensuring a seamless user experience (UX) through intuitive navigation and responsive design. Frontend development primarily uses three core technologies: HTML (for structuring content), CSS (for styling and layout), and JavaScript (for adding interactivity).
